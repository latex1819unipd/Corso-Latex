\section{Opzioni}
\begin{frame}{Opzioni}

È possibile sceglie la tipologia di documento più adatta alle nostre esigenze scegliendo 
tra le differenti opzioni di \texttt{documentclass}:
\mintinline{latex}{\documentclass[opzioni]{classe}}.

\begin{itemize}
	\item <2-> a4paper, letterpaper, etc
	\item <3-> twocolumn
	\item <4-> landscape
	\item <5-> draft
\end{itemize}

\end{frame}
